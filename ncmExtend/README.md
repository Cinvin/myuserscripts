进入项目目录：
```
cd ncmExtend
```

安装依赖：
```
pnpm install
```

构建 userscript：
```
pnpm run build
```

运行 ESLint 检查：
```
pnpm run lint
```

自动修复可修复的 ESLint 问题：
```
pnpm run lint:fix
```

检查 Prettier 格式是否一致：
```
pnpm run format
```

按 Prettier 规则自动格式化：
```
pnpm run format:write
```

use [vite-plugin-monkey](https://github.com/lisonge/vite-plugin-monkey)
