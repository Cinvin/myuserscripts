 ### 功能
试听时选择更高音质试听(支持超清母带,默认无损)  
个人主页:无需文件快速上传云盘歌曲(含周杰伦)、云盘匹配纠正、快速完成300首听歌量打卡任务、云盘导入导出  
 歌曲页:音乐下载(高音质下载,不消耗vip下载次数,文件不是ncm加密)、歌词下载、乐谱下载  
 ### 存在问题
云盘上传包含了大量咪咕24bit音源。该音源大多数是咪咕自己拿44.1kHz/16bit无损音质升频到48kHz/24bit的(为了推广5G)。它们大小在40M以上但是上传网易云后也不会识别为Hi-Res。拿spek看频谱会发现21kHz以上几乎没内容。这些假hires和无损音质相比只会占用空间。如果你在意这个可以不上传。目前不计划替换音源。
